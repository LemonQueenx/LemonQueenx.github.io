redux状态管理方案：
一：
1）全局combineReducer，createStore，通过顶层Provider向下传播。
2）使用 React Redux 的 connect() 方法来生成容器组件和 Redux store 连接起来。
3) 用同步请求自己手写实现异步请求。

二：
使用 applyMiddleware() 来增强 createStore()。

三：
反向注入reducer，节约性能。
