发表于 2020-09-07 | 分类于 3）前端 - 周边

whistle 是什么

whistle是基于Node实现的跨平台web调试代理工具，类似的工具有Windows平台上的Fiddler，主要用于查看、修改HTTP、HTTPS、Websocket的请求、响应，也可以作为HTTP代理服务器使用，不同于Fiddler通过断点修改请求响应的方式，whistle采用的是类似配置系统hosts的方式，一切操作都可以通过配置实现，支持域名、路径、正则表达式、通配符、通配路径等多种匹配方式，且可以通过Node模块扩展功能。

whistle 的直观优点

  1. 免费；
  2. 简便，安装启动操作都简便，界面友好，真机调试，扩展性好，社区活跃；
  3. 功能丰富，各大抓包工具应有的功能他都有，足够应对日常开发场景；
  4. 跨平台（Win/Mac/Linux）；

whistle使用方法

1. 安装 Node

whistle支持v0.10.0以上版本的Node，为获取更好的性能，推荐安装最新版本的Node。

2. 安装 whistle

Mac或Linux的非root用户需要在命令行前面加sudo，如：sudo npm install -g whistle

npm默认镜像是在国外，有时候安装速度很慢或者出现安装不了的情况，如果无法安装或者安装很慢，可以使用taobao的镜像安装：
