第一章 HTML

第一节 基础概念

HTML， Hyper Text Message Language，超文本标记语言，是前端的基础语言。

1. 基础结构

  • 最外层是<html>< /html>标签，里面分别是<head> < /head>和<body>< /body>标签，首尾呼应，成对出现，相互嵌套。

2. 简单的标签

<html></html>

lang属性：告诉搜索引擎爬虫，我们的网站是什么语言的。如：中文zh，英文en，德文de，法文fr等。
SEO 搜索引擎优化

<head></head>

head是头的意思，顾名思义，这里是放一些“头脑”性的东西。

<meta/>标签

1.charset字符集，现在一般都用UTF-8.以前还有gb2312，gbk，unicode等。
2.keywords网站的关键字，便于网络爬虫检索。
3.description描述，简介作用同上。
4.cache-control缓存控制，content的值可以是：Public 、 Private 、 no-Cache 、 no-Store。
5.expires期限，用于设定网页的到期时间。一旦网页过期，必须到服务器上重新传输。
6.pragma编译，用于设定禁止浏览器从本地机的缓存中调阅页面内容，设定后一旦离开网页就无法从Cache中再调出。
7.viewport视口，宽度等于设备宽度，初始视角是1倍视角，最大视角是1倍视角，最小视角是1倍视角，用户是否可以伸缩网页不可以，视角尺寸是覆盖。
8.X-UA-Compatible兼容IE，如果IE有安装Google Chrome Frame，那么就走安装的组件，如果没有，Edge 模式通知 Windows Internet Explorer以最高级别的可用模式显示内容。
9.HandheldFriendly手持式友好，针对手持设备优化，主要是针对一些老的不识别viewport的浏览器，比如黑莓
10.format-detection格式监测，禁止将页面中的一连串数字识别为电话号码并设置为手机可拨打的链接。告诉设备不识别邮箱，点击之后不自动发送。

<meta\/>标签是个有“头脑”的标签，还有更所设置，需要时可以进一步探索呦~

<title></title>

title是标题的意思，很简单。例如：


<p></p>

p, paragraph（段落），段落标签：独占一行

h1 ~ h6

h， headline（大字标题）， 标题标签:大小 加粗 独占一行

结构化标签

<span></span>

<div></div>

不建议使用的标签

<strong>< /strong>加粗标签
<em>< /em>斜体标签

<address>< /address>地址标签
<del>< /del>中划线（删除）标签

  • 因为，我们有一个原则：html就是结构，css是样式，js的交互。这种实现样式的标签，我们完全可以通过css操作不同标签来实现。

HTML编码

文字分隔符 &nbsp ;
小于号 &lt ;
大于号 &gt ;

3. 常用的标签

<a></a>

  • href : hyperText reference 超文本引用
    1、URL（相对路径，绝对路径，链接）baidu
    2、锚点 ↓
    3、协议限定符点我
    4、tel & mailTo 123456 123456@789.com

  • target属性
    _self（默认值）
    _blank
    _parent
    _top


<img/>

src: source 资源


<ul><li></li></ul>

ul: unordered lists 无序列表
ol: ordered lists 有序列表
li: list item 列表项

  • type属性，规定列表的项目符号的类型。不赞成使用。请使用样式取代它。

<table><tr><td>…</td></tr></table>

简单的 HTML 表格由 table 元素以及一个或多个 tr、th 或 td 元素组成。
tr 元素定义表格行，th 元素定义表头，td 元素定义表格单元。
更复杂的 HTML 表格也可能包括 caption、col、colgroup、thead、tfoot 以及 tbody 元素。
