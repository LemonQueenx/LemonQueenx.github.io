正则表达式 RegExp（Regular Expression）（1）

正则表达式是匹配特殊字符或有特殊搭配原则的字符的最佳选择。

一、创建方式：

     1.直接量 (个人推荐直接量)
           如：var reg = /^\d | \d$/;
     2.new RegExp();
           如：var reg = new RegExp(‘/^\\d | \\d$/‘);
        注：①js字符串不许出现回车符号，但是“\”代表转义字符。
           如：var a = “aaa\
                             bbb”;
           这样就不会报错了。
         ②“\”代表转义字符，“\\”代表\。“\n”代表换行

二、基本语法：

RegExp对象标识正则表达式，是对字符转执行 模式匹配 的强大工具。
1.compile     编译正则表达式
  exec          检索字符串中指定的值，返回找到的值，并确定其位置
  test           检索字符串中指定的值，返回true或false
2.i                代表不区分 大小写 （执行对大小写不敏感的匹配）
  g               代表 全局 匹配(查找所有匹配，而非找到第一个匹配后停止)
  m              多行 匹配
3.支持正则表达式的String对象的方法：
  search       检索与正则表达式相匹配的值
  match       找到一个或多个正则表达式的匹配
  replace      替换与正则表达式匹配的子串
  split           把字符串分割为字符串数组
