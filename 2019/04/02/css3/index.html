css3是最新的 CSS 标准。

css3常用新属性


兼容前缀浏览器
-webkitchrome和safari
-mozfireFox
-msIE
-oopera

border-radius 圆角

font-face 字体


可以直接用font-family:’ziti’了，字体资源一般设计都会提供，也可以在网络上找到。推荐网址：http://www.w3cplus.com/content/css3-font-face 和 https:://www.dafont.com

rgba() 颜色值

Violet green blue alpha

Gradient 渐变的背景色

1.linear-gradient 线性渐变 to
linear-gradient( [ direction ] , color1 [ precent / pixel ] ,color2 [ precent / pixel ] );
direction:
    1) 0 ~ 360deg角度
    2) to top/bottom/left/right
        to top left
        to bottom left…

eg:
    background:linear-gradient(50deg , red 10px,yellow 240px);

    background:linear-gradient(to right , SpringGreen , MintCream 90%);


2.radial=gradient 径向渐变 at
radial-gradient(shap at position , color1 [percent/pixel] , color2 [precent/pixel]);
    shap放射形状:circle,ellipse
    postion：center（圆心位置）、坐标值等

eg:
    radial-gradient(circle at center , rgba(96,0,190) , rgba(101,160,255))

    radial-gradient(ellipse at 200px 80px , rgba(96,0,190) , rgba(101,160,255))


background 背景


一、background-origin 背景图片起始位置
background-origin : border-box / padding-box / content-box;
background-origin : border-box;

background-origin : padding-box;

background-origin : content-box;

* 此处为了更清楚的看出背景图片的起始位置，我给背景图片设置了background-repeat：no-repeat。 二、background-size 背景图片尺寸
background-size : auto / <长度值> / <百分比> / cover / contain ;

三、background-position 背景图片位置
background-position:0px 10px;

四、background-repeat 背景图片是否重复
background-repeat:inherit;

border-image 边界图片（应用背景）


border-image:url(xxx.png) number style;
style:
    stretch 拉伸
    repeat 重复（会切割）
    round 铺满（不会切割）

eg:
    border-image:url(“图片”) 200 stretch;


box-shadow 盒子阴影


border-shadow : X轴偏移量 Y轴偏移量 [阴影模糊半径][阴影扩展半径][阴影颜色][投影方式：outset/inset];

text-shadow 文本阴影


text-shadow : X轴偏移量 Y轴偏移量 [阴影模糊半径] [阴影颜色];

word-wrap 文字边界换行


值：
    normal           只在允许的断字点换行（浏览器保持默认处理）。
    break-word    在长单词或 URL 地址内部进行换行。

background-clip 裁剪背景图片


background-clip : border-box / padding-box / content-box;
background-clip : border-box;

background-clip : padding-box;

background-clip : content-box;

* 注：webkit独有属性，且必须配合-webkit-text-fill-color:transparent;使用。有需要的时候可以深入学习。

css3选择器

属性选择器

E [ attr ^= “ val “ ]
E [ attr $= “ val “ ]
E [ attr *= “ val “ ]

初级伪类选择器 伪类用于向某些选择器添加特殊效果

:root 根标签选择器

:root{background:orange;}
:html{background:orange;}
二者等同

:not 否定选择器

div:not([class=”demo”]){background:orange;}

:empty 空标签选择器 （空格也算内容，伪元素不算）

p:empty{background:orange;}

target 目标元素选择器

用来匹配被location.hash选中的元素（即锚点元素）选择器可用于选取当前活动的目标元素。

伪类选择器

:first-child
:last-child
:nth-child(){},n代表自然数
    eg : ul li:nth-child(1){ul的第一个li，其中括号里还可以是3n+1这种表达式}
:nth-last-child(){}

:first-of-type
:last-of-type
:nth-of-type(){}
:nth-last-of-type(){}

:only-child 唯一子元素选择器
    选择是独生子的子元素，即该子元素不能有兄弟元素，它的父元素只有他一个直接子元素。
:only-of-type 类唯一选择器

:enabled 可用的元素
:disabled 不可用的元素，如web表单，输入框等

:checked选择框的被选中状态

:read-only 只读元素选择器
:read-write 非只读元素选择器（读写）
    eg :< input type=”text” readonly=”readonly”/>
          < input type=”text”/>

伪元素选择器

伪元素的效果是通过添加一个实际的元素才能达到的。
